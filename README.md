## 🧪 What is Rust-mol-viewer?

Rust-mol-viewer is a tool for looking at molecules in three dimensions. It shows detailed shapes of proteins and other molecules using 3D graphics. This helps users understand molecular structures better. The software is made using Rust, a programming language known for speed and safety.  

It uses modern graphics technology to run smoothly on your computer. You can load common molecular file formats like PDB and CIF. The tool also supports virtual reality (VR), so you can explore molecules in a more natural way if you have the right hardware.  

Rust-mol-viewer is designed to run on Windows computers. It works well even on average machines with a dedicated graphics card.

## 🛠️ Troubleshooting  

- **Program does not start:** Check if your system meets the requirements. Make sure Windows is updated.  

- **Cannot open molecular files:** Verify you use supported file formats like PDB or CIF. Check if the file is corrupted by opening it in a different viewer.  

- **Graphics issues (screen flickering, freezes):** Update your video card drivers. Restart your computer. Avoid running in remote desktop sessions.  

- **VR mode not working:** Confirm you have OpenXR compatible hardware and software installed. Restart the headset and Rust-mol-viewer.
